# CAPTCHA-CNN Security Prototype — Run Instructions

## Step 1 — Install everything (open this folder in VS Code, open terminal)
pip install -r requirements.txt

## Step 2 — Generate your dataset (takes a few minutes)
python generate_dataset.py

## Step 3 — Train and evaluate the CNN (takes 15-45 mins on a normal laptop CPU)
python train_model.py

This will print your REAL accuracy, precision, recall, and F1 score at the end,
and save:
- results.txt (copy these numbers into Chapter Four, Table 4.1)
- accuracy_graph.png
- loss_graph.png
- captcha_cnn_model.h5 (your trained model)

## Step 4 — Run the web app
python app.py

Then open your browser and go to: http://127.0.0.1:5000

## Step 5 — Take your 4 screenshots
1. Login page — just open the site (Figure 4.1)
2. Type the CAPTCHA correctly — screenshot the "Access Granted" page (Figure 4.2)
3. Type the CAPTCHA wrong on purpose — screenshot the "Access Denied" page (Figure 4.3)
4. On the CAPTCHA page, click "Simulate Bot Attack" link — screenshot the
   "Access Temporarily Blocked" page (Figure 4.4)

## Notes
- If pip install fails on tensorflow, try: pip install tensorflow-cpu
- If port 5000 is busy, change app.run(debug=True) to app.run(debug=True, port=5001)
- All numbers you get from results.txt are REAL — generated by your own trained
  model, not made up. Safe to defend in front of your supervisor.
